# Restaurant Rating Prediction

## Objective
Build a Machine Learning model to predict restaurant ratings based on various restaurant features.

## Dataset
Restaurant dataset containing:
- City
- Cuisines
- Votes
- Average Cost for Two
- Online Delivery
- Location Information
- Aggregate Rating

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Model
Decision Tree Regressor

## Results

MAE : 0.266

MSE : 0.174

R² Score : 0.924

## Key Findings

- Votes were the most important feature.
- Location and cuisines had moderate impact.
- The model achieved high prediction accuracy.
